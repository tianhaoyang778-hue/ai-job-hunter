# parsers/

每个文件对应一家公司的官网爬取逻辑，供 Step 2–4 使用。

## 使用方式

执行 skill 时，若目标公司在此文件夹有对应文件，**优先读取**，不要重新摸索。
文件包含：已验证的 URL 模板、URL 参数说明、翻页方式、已知的反爬行为。

## 新增一家公司

首次跑通某公司后，将以下内容整理成 `公司英文名.md` 存入此文件夹：

1. 有效的招聘页 URL 模板（含参数说明）
2. 城市代码表（如有）
3. 翻页方式（URL 参数 / 点击按钮 / JS 注入）
4. 搜索框操作方式（URL 参数直达 / form_input / JS 设值）
5. 已知的访问限制或反爬行为
6. 详情页是否需要登录
