# 🎯 AI Job Hunter

> 用自然语言告诉 Claude 你想找什么岗位、你的背景是什么，自动抓取招聘官网并生成带个性化评级的 Excel 表格。

---

## 能做什么

- 自动访问大厂招聘官网（字节跳动等），按关键词搜索岗位
- 逐一读取每个岗位的详情页原文（职责 + 要求），不依赖列表页摘要
- 根据你的个人背景对每个岗位评级：🟢 高 / 🟡 中 / 🔴 低
- 输出带颜色评级、匹配说明、详情页链接的 Excel 表格，可直接用于投递决策

**支持任意公司、任意岗位类型**，不限制行业或背景。已内置字节跳动的爬取逻辑，其他公司首次运行后会自动学习并记录。

---

## 环境要求

| 环境 | 支持程度 | 说明 |
|------|---------|------|
| **Claude Cowork** | ✅ 完整功能 | 有浏览器 Agent，可直接访问招聘官网 |
| Claude Code | ⚠️ 功能受限 | 大厂官网有反爬保护，可能无法访问 |
| Claude.ai 网页版 | ⚠️ 功能受限 | 同上 |

**强烈推荐在 Claude Cowork 中使用。** 大厂招聘官网（如 jobs.bytedance.com）有反爬保护，只有 Cowork 的浏览器 Agent 能稳定访问并读取完整内容。

## 安装方法

### ⭐ 方法一：下载 .skill 文件安装（推荐）

最简单的方式，适合所有人，无需任何技术背景。

1. 前往本仓库的 [Releases](../../releases) 页面
2. 下载最新版本的 `ai-job-hunter.skill` 文件
3. 打开 Claude Cowork
4. 将 `.skill` 文件拖入 Claude 的对话框，或通过设置页面的「安装 Skill」入口上传
5. 安装成功后，直接对话触发即可

### 方法二：克隆仓库手动安装

适合想查看或修改源文件的用户。

```bash
git clone https://github.com/你的用户名/ai-job-hunter.git
```

克隆后将整个 `ai-job-hunter/` 文件夹放入 Claude 的 skills 目录：

- **Cowork / Claude Code**：`~/.claude/skills/user/ai-job-hunter/`

### 方法三：只下载 SKILL.md（最简安装）

如果你只想快速试用核心功能：

1. 下载本仓库的 `SKILL.md` 文件
2. 在 skills 目录下新建文件夹 `ai-job-hunter/`
3. 将 `SKILL.md` 放入其中

> ⚠️ 此方式缺少 `scripts/generate_excel.py`，Excel 生成步骤会降级为内联代码执行，稳定性较低。同时缺少字节跳动等公司的内置爬取逻辑，首次运行需要自行摸索。不推荐长期使用。

## 使用方式

安装后，在 Claude Cowork 中直接说：

```
帮我搜索字节跳动的AI产品实习
腾讯有没有运营方向的实习？
给我找适合文科背景的互联网实习
整理一份阿里巴巴的数据分析岗位表格
```

Claude 会自动触发此 skill，引导你填写岗位关键词和个人背景，然后开始执行。

### 运行模式选择

| 模式 | 适合场景 | 说明 |
|------|---------|------|
| **快速模式** | 第一次试用、验证流程 | 只抓前 5 页（约 150 个岗位），几分钟出结果 |
| **细致模式** | 正式求职、要求覆盖全面 | 每 5 页预览确认一次，最终合并为完整 Excel |

## 评级逻辑说明

评级完全基于你在对话中描述的个人背景，没有固定标准，对任何行业和方向都适用。

skill 内置了一份真实用户背景的评级示范（`prompts/scoring/example-mine.md`）：

> 背景：文科，AI native，熟练使用 AI 工具（workflow / 产品 demo / Prompt 调优），不写代码，不做模型训练，偏应用方向

这份示范的作用是告诉 Claude「一套好的评级标准长什么样」。每次运行时，Claude 会参考这个结构，根据你自己描述的背景重新生成专属评级标准，而不是直接套用示范内容。

## 常见问题

**Q：为什么一定要用 Cowork？**
大厂招聘官网有反爬保护，WebFetch 等普通网络工具会被拦截。Cowork 的浏览器 Agent 模拟真实浏览器操作，可以稳定访问。Claude Code 和 Claude.ai 网页版在遇到拦截时会降级为搜索引擎结果，数据完整性无法保证。

**Q：需要登录招聘官网吗？**
大部分岗位的详情页不需要登录即可访问。如果遇到需要登录的情况，Claude 会暂停并提示你在浏览器中完成登录后继续。

**Q：搜索结果很多，会全部抓取吗？**
快速模式只抓前 5 页（约 150 个岗位）。细致模式可以抓全部，每 5 页暂停让你确认是否继续，防止运行时间过长。

**Q：抓取失败的岗位怎么处理？**
Claude 会在生成 Excel 前列出所有失败的岗位和链接，你可以手动打开链接，将内容复制给 Claude 补充进表格。

**Q：支持社会招聘吗？**
目前主要针对校园招聘和日常实习。社会招聘页面结构不同，部分公司可能需要调整 URL 参数，但整体流程是通用的。
