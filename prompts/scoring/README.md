# prompts/scoring/

存放评级 prompt 的参考模板。

## Claude 使用说明

**每次执行 Step 6 评级前，必须读取本文件夹。**

### 使用流程

1. 读取 `example-mine.md`，理解一份好的评级 prompt 的结构和逻辑
2. 对照用户在 Step 1 提供的个人背景，按相同结构重新生成适合该用户的评级标准
3. 用新生成的标准对所有岗位评级，**不要直接套用 `example-mine.md` 的内容**

### 关键原则

- 用户背景决定一切：同一个岗位，对不同背景的人评级可能完全相反
- 用户明确说"不想做X"，等同于硬性排除，直接标低
- 技术硬门槛（如"必须会写代码"）对非技术背景用户是一票否决
- 岗位不一定与 AI 相关，评级标准要根据用户的实际目标领域调整
